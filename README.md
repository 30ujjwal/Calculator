# Simple Calculator Application

A simple and user-friendly calculator application built to perform basic arithmetic and advanced operations.

## ✨ Features
* **Addition (+):** Adds two numbers.
* **Subtraction (-):** Subtracts the second number from the first.
* **Multiplication (*):** Multiplies two numbers.
* **Division (/):** Divides numbers with error handling for division by zero.
* **Square Root (√):** Calculates the square root of a number.

## 🚀 Technologies Used
*  IntelliJ IDEA 

## 💻 How to Run
1. Clone this repository or download the files.
2. Open the project in your preferred IDE.
3. Run the main file to start using the calculator.
